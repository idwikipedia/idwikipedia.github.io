3. Server side script biasanya hanya ada pada web…​


Server side script tidak hanya terbatas pada web. Server side script merujuk pada skrip atau kode yang dieksekusi oleh server untuk menghasilkan respon yang akan dikirim kembali ke klien (pengguna). Ini berarti bahwa server side script dapat digunakan dalam berbagai jenis aplikasi yang melibatkan interaksi antara server dan klien.

Meskipun server side script umumnya dikaitkan dengan pengembangan web, seperti PHP, Python, atau Node.js, mereka juga dapat digunakan dalam aplikasi lain seperti aplikasi perangkat lunak berbasis server, layanan web, atau sistem basis data.

Dalam konteks web, server side script sering digunakan untuk memproses data yang dikirim dari klien, mengakses basis data, membangun halaman dinamis, atau menyediakan fungsi dan layanan lainnya kepada pengguna. Skrip ini dijalankan di sisi server dan hasilnya dikirim ke klien dalam bentuk HTML, JSON, XML, atau format lainnya yang dapat ditampilkan oleh browser atau aplikasi klien.

Namun, penting untuk dicatat bahwa ada juga client side scripting, yang merujuk pada kode yang dieksekusi oleh browser atau aplikasi klien di sisi pengguna. JavaScript adalah bahasa pemrograman client side scripting yang umum digunakan dalam pengembangan web untuk memberikan interaktivitas dan fungsionalitas pada halaman web.

13. pengertian aplikasi client server serta contoh dan penjelasannya


Client-Server adalah pembagian kerja antara server dan client yang mengakses server dalam suatu jaringan.

Contohnya:

1. File Servers

File server vendors mengklaim bahwa mereka pertama menemukan istilah client-server.

Untuk sharing file melalui jaringan

2. Database Servers

Client mengirimkan SQL requests sebagai pesan pada database server,selanjutnya hasil perintah SQL dikembalikan.

Server menggunakan kekuatan proses yang diinginkan untuk menemukan data yang diminta dan kemudian semua record dikembalikan pada client.

3. Transaction Servers (Transaksi Server)

Client meminta remote procedures yang terletak pada server dengan sebuah SQL database engine.

Remote procedures ini mengeksekusi sebuah grup dari SQL statement

Hanya satu permintaan / jawaban yang dibutuhkan untuk melakukan transaksi

4. Groupsware Servers

Dikenal sebagai Computer-supported cooperative working

Manajemen semi-struktur informasi seperti teks, image, , bulletin boards dan aliaran kerja

Data diatur sebagai dokumen

5. Object Application Servers

Aplikasi client/server ditulis sebagai satu set objek komunikasi

Client objects berkomunikasi dengan server objects melalui Object Request Broker (ORB)

Client meminta sebuah method pada remote object

6. Web Application Servers (Aplikasi Web Servers)

World Wide Web adalah aplikasi client server yang pertama yang digunakan untuk web.

Client dan servers berkomunikasi menggunakan RPC seperti protokol yang disebut HTTP.

24. 3. Jelaskan pengertian virtual private server (vps) beserta contohnya !


Virtual private server (VPS) adalah sebuah teknologi yang memungkinkan sebuah fisik server dibagi menjadi beberapa bagian virtual yang dapat diakses dan digunakan secara independen oleh beberapa pengguna. Dengan menggunakan VPS, setiap pengguna dapat memiliki akses ke sistem operasi dan resourcenya sendiri, seperti RAM, CPU, dan storage, sehingga dapat menjalankan aplikasi dan mengelola data sesuai kebutuhan mereka.

Contoh penggunaan VPS adalah ketika sebuah perusahaan ingin menjalankan beberapa aplikasi yang memerlukan sistem operasi dan resource yang berbeda-beda. Dengan menggunakan VPS, perusahaan tersebut dapat membagi satu fisik server menjadi beberapa bagian virtual yang dapat diakses oleh masing-masing aplikasi, sehingga aplikasi tersebut dapat berjalan secara terpisah dan tidak terpengaruh oleh aktivitas dari aplikasi lain. Selain itu, VPS juga dapat digunakan oleh individu atau perusahaan kecil yang ingin memiliki kontrol penuh atas server virtualnya tanpa perlu membeli dan mengelola fisik server sendiri.

31. pengertian dan contoh hardware server dan brainware​


Jawaban:

Hardware adalah perangkat keras dari sebuah komputer yang dapat disentuh dan dipindahkan. Perangkat keras terdiri dari empat bagian yakni, perangkat bagian masukan (input), perangkat bagian keluaran (output), perangkat bagian pemrosesan (processing), dan perangkat bagian penyimpanan (storage). Contoh dari perangkat bagian input adalah mouse, keyboard, microphone, dan touch screen. Contoh dari perangkat bagian output adalah monitor, speaker, dan printer. Contoh dari perangkat bagian pemrosesan (processing) adalah prosesor yang berfungsi sebagai otak dari sebuah komputer. Contoh perangkat bagian penyimpanan (storage) adalah flashdisk dan harddisk.

Server adalah sebuah komputer yang menyediakan layanan untuk komputer lainnya dalam jaringan. Server dapat berupa perangkat keras atau perangkat lunak. Contoh dari server hardware adalah server blade, server tower, dan server rack.

Brainware adalah pengguna atau orang yang menggunakan sistem komputer. Tanpa adanya brainware, maka hardware dan software tidak akan bisa bekerja.
