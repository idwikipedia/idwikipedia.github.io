ENERGI PADA KAPASITOR

Kapasitas kapasitor memiliki arti kemampuan kapasitor untuk menyimpan muatan listrik. Kapasitas kapasitor disebut juga kapasitansi. Persamaan kapasitas kapasitor

[tex]C \:=\: \frac{q}{V}[/tex]

Kapasitor dapat menyimpan energi dalam bentuk medan listrik. Persamaan untuk menentukan energi pada kapasitor adalah:

[tex]W \:=\: \frac{1}{2} \frac{q^2}{C}[/tex][tex]W \:=\: \frac{1}{2} qV[/tex][tex]W \:=\: \frac{1}{2} CV^2[/tex]

dengan

q = jumlah muatan (C)C = kapasitas kapasitor (F)V = beda potensial (V)W = energi listrik (J)

1 F = [tex]10^6[/tex] μF

1 μF = [tex]10^{-6}[/tex] F

1pF = [tex]10^{- 12}[/tex] F

Diketahui:

V = 12 VoltW = [tex]3,6 \times 10^{- 6}[/tex]

Ditanyakan:

C?

Penjelasan dengan langkah-langkah:

[tex]W \:=\: \frac{1}{2} CV^2[/tex]

[tex]2W \:=\: CV^2[/tex]

[tex]CV^2 \:=\: 2W[/tex]

[tex]C \:=\: \frac{2W}{V^2}[/tex]

[tex]C \:=\: \frac{2 \times 3,6 \times 10^{- 6}}{12^2}[/tex]

[tex]C \:=\: \frac{7,2 \times 10^{- 6}}{144}[/tex]

[tex]C \:=\: \frac{7,2}{144} \times 10^{- 6}[/tex]

[tex]C \:=\: 0,05 \times 10^{- 6}[/tex]

C = 0,05 μF

[tex]C \:=\: 5 \times 10^{- 2} \times 10^{- 6}[/tex]

C = 5 × 10⁻⁸ F

Kapasitas kapasitor yang dipasang adalah 0,05 μF.

[tex]\displaystyle \frac{1}{C_e} = \frac{1}{C_1} + \frac{1}{C_2}\\\frac{1}{C_e} = \frac{2}{10^{-11}~F}\\C_e = \frac{1}{2}\times10^{-11} ~F\\\therefore~~ C_e = \boxed{0.5\times10^{-11}~F}[/tex]

18. 2 kapasitor di susun seri yang besarnya kapasitor pertama 30 mf dan kapasitor kedua 290 mf kemudian dengan baterai 6 volt Tentukanlah: a. kapasitas kapasitor rangkaian b. muatan tiap² kapasitor c. tegangan pada tiap² kapasitortolong dijawab mau di kumpul sebentar​


Jawaban:

**Jawaban:**

**a. Kapasitas kapasitor rangkaian**

Kapasitas kapasitor rangkaian dirumuskan sebagai berikut:

```

C = C1 * C2 / (C1 + C2)

```

Keterangan:

* C = Kapasitas kapasitor rangkaian (μF)

* C1 = Kapasitas kapasitor pertama (μF)

* C2 = Kapasitas kapasitor kedua (μF)

Pada soal ini, C1 = 30 μF dan C2 = 290 μF, sehingga kapasitas kapasitor rangkaian adalah:

```

C = 30 μF * 290 μF / (30 μF + 290 μF)

= 29.7 μF

```

**b. Muatan tiap² kapasitor**

Muatan tiap kapasitor dirumuskan sebagai berikut:

```

Q = C * V

```

Keterangan:

* Q = Muatan kapasitor (Coulomb)

* C = Kapasitas kapasitor (μF)

* V = Tegangan kapasitor (volt)

Pada soal ini, C = 29.7 μF dan V = 6 volt, sehingga muatan tiap kapasitor adalah:

```

Q1 = C1 * V

= 30 μF * 6 volt

= 180 μC

```

```

Q2 = C2 * V

= 290 μF * 6 volt

= 1740 μC

```

**c. Tegangan pada tiap² kapasitor**

Tegangan pada tiap kapasitor dirumuskan sebagai berikut:

```

V = Q / C

```

Keterangan:

* V = Tegangan kapasitor (volt)

* Q = Muatan kapasitor (Coulomb)

* C = Kapasitas kapasitor (μF)

Pada soal ini, Q1 = 180 μC dan C1 = 30 μF, sehingga tegangan pada kapasitor pertama adalah:

```

V1 = Q1 / C1

= 180 μC / 30 μF

= 6 volt

```

```

V2 = Q2 / C2

= 1740 μC / 290 μF

= 6 volt

```

**Jawaban lengkap:**

a. Kapasitas kapasitor rangkaian adalah 29.7 μF.

b. Muatan tiap kapasitor adalah 180 μC untuk kapasitor pertama dan 1740 μC untuk kapasitor kedua.

c. Tegangan pada tiap kapasitor adalah 6 volt.

**Penjelasan:**

* Pada kapasitor seri, kapasitas kapasitor total adalah hasil bagi dari kapasitas dua kapasitor.

* Muatan pada kapasitor seri sama, karena muatan tidak dapat mengalir melalui kapasitor.

* Tegangan pada kapasitor seri tidak sama, karena tegangan kapasitor berbanding terbalik dengan kapasitas kapasitor.

36. tolong dijawab soal fisika tentang kapasitor ini


Pada Rangkaian Capasitansi.

Ketika beberapa capasitor di rangkai secara seri.
1/Cs = 1/C + 1/C +...
1/Cs = n 1/C
Cs = C/n

Ketika n capasitor dirangkai secara paralel.
Cp = C + C +...
Cp = nC

Kemudian rangkaian seri dan paralel di rangkai secara seri

1/C = 1/Cp + 1/Cs

1/C = 1/nC + n/C
1/C = (1/nC + n^2/nC)
1/C = (1+ n^2) / nC

C = nC / (1 + n^2)

Jadi, capasitor penggantinya adalah nC/(1+n^2)
