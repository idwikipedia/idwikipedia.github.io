12. Bilangan berpangkat tiga adalah perkalian berulang sebanyak tiga kali. Misal : Bilangan 4 pangkat 3 = 4³ = 4 × 4 × 4 = 64 Bilangan 4 disebut dengan pokok. Bilangan 3 disebut dengan pangkat. Bilangan 64 disebut bilangan kubik atau hasil bilangan pangkat tiga.​


Jawab:

Bilangan 64 disebut hasil dari bilangan pangkat 3

Penjelasan dengan langkah-langkah:

Mengapa hasil dari pangkat 3 ?

⇒ karena, pada soal yang ditanya adalah bilangan pangkat 3. kubik hanya dipakai saat mencari volume, dan ada satuan panjangnya ( misal : km, hm, dam, m, dm, cm, mm )

====================================================

Berikut adalah contoh bilangan yang berpangkat :

2³ = 2 × 2 × 2

Maksudnya adalah angka dua dikalikan dengan angka yang sama sebanyak tiga kali dan menghasilkan bilangan dua pangkat tiga.

Contoh Perpangkatan 2:

2² = ( 2 x 2 ) = 4

3² = ( 3 x 3 ) = 9

4² = ( 4 x 4 ) = 16

5² = ( 5 x 5 ) = 25

Contoh Perpangkatan 3:

2³ = ( 2 x 2 x 2 ) = 8

3³ = ( 3 x 3 x 3 ) = 27

4³ = ( 4 x 4 x 4 ) = 64

5³ = ( 5 x 5 x 5 ) = 125

==================================

22. personal kan penyebut pecahan-pecahan berikut tiga dibagi pangkat 5​


Jawab:

[tex]\dfrac{3}{\sqrt{5}}=\bf\dfrac{3\sqrt{5}}{5}=\dfrac{3}{5}\sqrt{5}[/tex]
 

Penjelasan dengan langkah-langkah:

Merasionalkan penyebut (bukan mempersonalkan penyebut ya..)Dan juga penyebutnya bukan “pangkat 5”, tapi “akar 5”.

(Jika tidak sesuai yang dimaksud, silahkan report atau berkomentar.)

[tex]\dfrac{3}{\sqrt{5}}[/tex] kita kalikan dengan pecahan lain yang bernilai 1 dan dapat membuat penyebut menjadi bilangan bulat, yaitu [tex]\dfrac{\sqrt{5}}{\sqrt{5}}[/tex].

[tex]\dfrac{3}{\sqrt{5}}\times\dfrac{\sqrt{5}}{\sqrt{5}}=\dfrac{3\sqrt{5}}{\sqrt{5}\times\sqrt{5}}[/tex]

[tex]\text{Maka:}\ \dfrac{3}{\sqrt{5}}=\bf\dfrac{3\sqrt{5}}{5}=\dfrac{3}{5}\sqrt{5}[/tex]

[tex]\sf \frac{3 \sqrt{5} }{5 } [/tex]

Penjelasan dengan langkah-langkah:

[tex] \sf = \frac{3}{ \sqrt{5} } [/tex]

[tex]\sf = \frac{3}{ \sqrt{5} } \times \frac{ \sqrt{5} }{ \sqrt{5} } [/tex]

[tex]\sf = \frac{3 \sqrt{5} }{ \sqrt{5} \sqrt{5} } [/tex]

[tex]\sf = \frac{3 \sqrt{5} }{5 } [/tex]

[tex]\sf = \frac{ 3\sqrt{5} }{ \sqrt{ {5}^{2} } } [/tex]
