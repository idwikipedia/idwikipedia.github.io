9. Berdasarkan sumbernya, software terbagi menjadi dua jenis, Open Source Software, Closed Source Software atau proprietary software. Jelaskan apa yg dimaksud Open Source Software, Closed Source Software atau proprietary software dan berikan Contohnya..​


Jawaban:

Perbedaan yang dapat dilihat dari Open Source Software dan Closed Source Software atau Proprietary Software, yaitu:

Open Source adalah sistem operasi yang membuka/membebaskan source code-nya untuk dilihat oleh orang lain dan membiarkan orang lain mengetahui cara kerja software tersebut dan sekaligus

Contoh Open Source Software adalah Android dan Linuxmemperbaiki kelemahan-kelemahan yang ada pada software tersebut.

Closed Source atau Proprietary Source adalah sistem operasi yang dikembangkan secara internal oleh seseorang, kelompok atau perusahaan tertentu dan sistem operasi ini tidak membuka secara umum kode programnya.

Contoh Closed Source atau Prorietary Source Software adalah Windows dan MacOs

Penjelasan:

Berikut pembahasan terhadap perbedaan Open Source Software dan Closed Source Software:

Open Source Software merupakan sistem operasi atau software yang mengizinkan pengguna atau orang selain developer (pembuat) untuk menyalin, memodifikasi atau bahkan menghapus source code sesuai dengan keinginan pengguna. Sistem operasi Open Source Software ini menyediakan pengembangan yang bebas bagi penggunanya. Hal ini dapat diterapkan karena akses untuk source code sistem operasi dapat dibuka dan dilihat, sehingga bagi kalian yang hanya sekedar melihat source code pada software yang digunakan atau suka dengan memodifikasi suatu sofware dapat menumpahkan kreatifitas tanpa perlu takut menyalahi lisensi yang sudah dibuat, tentu dengan menyesuaikan aturan dan hukum yang ada agar tidak digunakan untuk hal-hal yang negatif.

Contoh Open Source Software adalah Android dan Linux

Sementara Closed Source atau Prorietary Source Software merupakan sistem operasi yang yang source code-nya tidak dibuka untuk umum. Closed Source sistem operasi juga merupakan sistem operasi yang mempunyai lisensi yang artinya hanya dapat dikembangkan oleh pemilik atau developer terkait dan kebanyakan untuk mendapatkan atau menggunakannya dikenakan biaya (berbayar). Source code pada Closed Source tidak dapat dilihat atau dirubah sehingga pengguna atau orang lain yang tidak membeli atau bahkan orang yang sudah membeli software ini tidak diperbolehkan untuk menyalin, mengubah, hingga menghapus source code-nya

Contoh Closed Source atau Prorietary Source Software adalah Windows dan MacOs

Jawaban: D. SOFTWARE UTILITY

Penjelasan:

Antivirus merupakan aplikasi yang sudah dapat dipastikan terpasang pada setiap sistem operasi (terutama Windows), sistem operasi tanpa Antivirus akan sangat beresiko tinggi terhadap kehilangan dan kerusakan data atau bahkan pencurian data sensitif dengan berbagai teknik hacking

Contoh perangkat lunak antivirus diantaranya AVG Antivirus, Microsoft Security Essentials, Avira, McAfee, Kaspersky, BitDefender, dan masih banyak lagi termasuk diantaranya antivirus lokal (buatan Indonesia) seperti SmadAV dan PCMAV.

Antivirus diatas merupakan perangkat lunak berbayar, artinya Anda harus membeli lisensi untuk dapat menggunakan perangkat lunak antivirus tersebut secara penuh (full), namun beberapa antivirus menyediakan versi free (gratis) dan versi trial (uji coba), seperti antivirus favorit Saya AVG Free Antivirus merupakan perangkat lunak gratis namun dengan beberapa batasan fitur jika dibandingkan dengan versi berbayarnya.

Keberadaan perangkat lunak antivirus ini sangatlah penting, jadi usahakan hindari menggunakan antivirus bajakan karena penggunaan antivirus bajakan (berlaku juga untuk perangkat lunak bajakan apapun) akan sangat membahayakan sistem bahkan data yang ada didalam komputer, antivirus bagaikan obat, bayangkan saja apa yang terjadi jika obat tersebut bajakan (palsu).

Selain itu gunakan juga antivirus lokal, seperti Saya yang menggunakan AVG Free Antivirus dan SmadAV Antivirus, keberadaan perangkat lunak SmadAV ini akan  sangat membantu kinerja antivirus utama (AVG) dalam menangani virus lokal, selain itu banyak fitur (salah satunya Smad-Lock) yang akan sangat berguna.


11. Berdasarkan fungsinya, software dikategorikan sebagai .... a. Application software dan System software b. System software dan Compilers c. System software dan Compilers d. System software dan assemblers


Berdasarkan fungsinya, software yang digunakan dalam komputer dikategorikan menjadi Application software dan System software

Pembahasan:

Dalam penggunaan komputer, kita membutuhkan software atau perangkat lunak agar bisa mengoperasikan komputer. Tanpa adanya perangkat lunak, maka suatu komputer tidak bisa digunakan secara maksimal. Dilihat dari fungsinya, software secara garis besar dibedakan menjadi dua macam yaitu application software dan system software.

Application software merupakan perangkat lunak yang memiliki fungsi dan tujuan khusus. Kekhususannya ini membuat perangkat lunak ini efektif dalam mengerjakan suatu tugas secara spesifik, namun secara general tidak bisa. Fungsi spesifik antara lain untuk pengolahan kata bisa menggunakan microsoft word, untuk pengolahan gambar bisa menggunaka Adobe photoshop, dan pengolah angka bisa menggunakan Microsoft exel.

System software merupakan perangkat lunak yang membuat suatu komputer bisa bekerja secara umum seperti menyimpan file, mengorganisit file, hingga menggandakan suatu file. System software juga menjadi tempat untuk instalasi dari application software. Contoh dari system software antara lain Microsoft Windows 10, Ubuntu 19.04, Machintos.

16. wablah soal-soal berikut! Jelaskan bentuk integrasi antara hardware, software dan brainware!​


Integrasi antara hardware, software, dan brainware merupakan elemen penting dalam teknologi komputer. Ketiga elemen ini bekerja bersama-sama untuk menciptakan sistem yang efektif dan fungsional.

1. Hardware:

  Hardware merujuk pada semua perangkat fisik dan komponen yang membentuk sistem komputer. Ini termasuk komputer, perangkat keras periferal seperti keyboard, mouse, monitor, printer, dan semua komponen elektronik di dalamnya seperti prosesor, memori, dan penyimpanan. Hardware menyediakan platform fisik yang diperlukan untuk menjalankan software.

2. Software:

  Software meliputi program dan instruksi yang mengatur fungsi sistem komputer. Ini mencakup sistem operasi, perangkat lunak aplikasi, perangkat lunak utilitas, dan perangkat lunak pengembang. Software memungkinkan penggunaan dan pengoperasian perangkat keras dengan menyediakan antarmuka yang mudah digunakan dan mengelola sumber daya perangkat keras.

3. Brainware:

  Brainware merujuk pada pengguna manusia dari sistem komputer, termasuk keahlian, pengetahuan, keterampilan, dan pengalaman. Brainware berperan penting dalam mengoperasikan dan memanfaatkan sistem komputer dengan efisien. Pengguna yang terampil dapat memaksimalkan potensi perangkat keras dan perangkat lunak yang ada.

Integrasi antara ketiga elemen ini terjadi ketika hardware dan software dirancang dengan mempertimbangkan kemampuan, kebutuhan, dan keterampilan pengguna manusia. Hardware yang baik harus kompatibel dengan software yang relevan, dan software harus dirancang dengan baik agar dapat digunakan dengan nyaman oleh pengguna. Selain itu, brainware yang efektif melibatkan pemahaman dan penggunaan optimal dari hardware dan software yang tersedia.

Dalam keseluruhan, integrasi antara hardware, software, dan brainware sangat penting dalam mencapai sistem komputer yang efisien, fungsional, dan bermanfaat bagi pengguna.

28. permasalahan pada software​ tolong di jawab besok di kumpul


Jawaban:

Macam-macam Permasalahan Pada Software

Macam-macam Permasalahan Software

Masalah software kadang bisa menjadi serius ketika kita salah memperbaikinya, untuk itulah di perlukan pemahaman yang lebih lanjut, sehingga setiap jenis kesalahan software bisa teratasi dengan baik. Permasalahan yang sering muncul pada bagian software ini pada umumnya bisa dikelompokkan atas:

macam-macam permasalahan software

a) Kesalahan setting konfigurasi jaringan

Kesalahan setting konfigurasi sering terjadi pada kartu jaringan yang menggunakan model ISA karena kita harus menentukan :

Alamat port I/O

Nomor Interupt

Direct Memory Access Request line

Buffer memory Address

Berbeda dengan kartu model ISA Kartu jaringan yang menggunakan model PCI tidak perlu mengeset karena secara otomatis telah tersedia.

b) Kesalahan Protocol yang digunakan

Hal ini sering terjadi pada kartu jaringan yang menggunakan slot ISA karena penentuan harus dilakukan secara manual. Apabila kita menggunakan protocol kartu jaringan model PCI hal tersebut jarang terjadi apabila kita telah menginstall driver dengan benar.

c) Kesalahan pengalamatan IP

Setiap komputer dalam suatu jaringan merupakan identifikasi alamat yang unik, sehingga tidak diperbolehkan ada alamat yang sama. IP Address dalam jaringan tidak diperbolehkan sama karena

merupakan identitas untuk masing-masing komputer dalam jaringan untuk komunikasi data, jika terjadi alamat yang sama maka kedua komputer tidak dapat mengakses jaringan karena terjadi perebutan nomor alamat tersebut.

d ) Kesalahan Indentifikasi Client dan server komputer

Penentuan antara komputer server dan komputer client harus jelas untuk jaringan client server, berbeda pada jaringan peer to peer tidak ada penentuan client dan server.

e) Kesalahan Service Network (file and print sharing)

Service network (file and print sharing) yang tidak aktif bisa dikarenakan file and print sharing yang kita hubungi sedang tidak aktif atau kita belum melakukan file and print sharing.

f) Kesalahan Security System

Kesalahan pemasukan password pada saat kita masuk dalam jaringan sehingga kita tidak dapat masuk dalam jaringan karena kesalahan pengamanan (password).

g) Kerusakan file program, sehingga perlu di update

Kerusakan file program yang menyebabkan sistem operasi tidak bisa berjalan atau menyebabkan kartu jaringan tidak dapat bekerja (tidak aktif).
