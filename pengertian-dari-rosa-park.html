25. Selisih uang Rosa dan Rita adalah Rp.1.500.000 dan 3 kali uang Rosa ditambahuang Rita adalah Rp. 13.500.000.Tentukanlah jumlah uang Rosa dan Rita ?​


Misalkan

Uang rosa = a

Uang rita = b

Maka, diperoleh persamaan berikut

a - b = 1.500.000

3a  + b = 13.500.000

Eliminasi kedua persamaan tsb

a - b = 1.500.000

3a + b = 13.500.000  +

4a = 15.000.000

 a = 15.000.000 / 4

 a = 3.750.000

a - b = 1.500.000

b = a - 1.500.000

b = 3.750.000 - 1.500.000

b = 2.250.000

Jadi, uang rosa = Rp 3.750.000,- dan uang rita = Rp 2.250.000,-
