Penjelasan:

Start jongkok merupakan salah satu teknik dasar yang terdapat dalam lari jarak pendek. Start jongkok digunakan karena dapat menghasilkan tolakan ketika melakukan awalan berlari. Lari jarak pendek merupakan olahraga lari dengan jarak 100 m, 200 m, dan 400 m. Faktor utama yang harus dimiliki oleh pelari jarak pendek ialah kecepatan. Dalam lari jarak pendek, terdapat beberapa teknik dasar yang harus dikuasai oleh setiap pemain. Teknik dasar dalam lari jarak pendek adalah.

1. Teknik start.

Dalam lari jarak pendek, teknik start yang digunakan adalah start jongkok. Start jongkok dilakukan dengan cara.

•Ketika mendengar aba-aba "bersedia" pemain melakukan persiapan dengan cara menaruh kedua kaki pada papan tumpuan, badan membungkuk, dan kedua tangan ditempatkan di belakang garis start.

• Ketika pemain mendengar aba-aba "siap" pemain memindahkan berat badan pada kaki depan, kedua lutut di angkat, dan pandangan fokus kedepan.

• Ketika pemain mendengar aba-aba "ya" atau bunyi pistol, pemain melakukan tolakan dan bersiap melakukan gerakan lari.

2. Posisi tubuh saat berlari.

Ketika berlari, pemain menggunakan langkah lebar dengan posisi badan sedikit condong ke depan. Ayunan tangan dilakukan dengan kuat hingga dibawah dagu. Gerakan lari dilakukan dengan tubuh rilex dan mata fokus ke depan.

3. Teknik memasuki garis finish.

Ketika akan memasuki garis finish, pemain menambah kecepatan berari. Pada saat memasuki garis finish, pemain harus menjatuhkan salah satu bahu ke depan untuk dapat memasuki garis finish terlebih dahulu.

Ketika sikap bersedia kita harus memposisikan kedua tangan dan jari kita ditengah , dan ketika aba-aba siap, bokong dinaikkan, dan ketika aba-aba Ya, maka kita harus mulai berlari yang dimulai dengan kaki kanan.

2.Macam Macam Start Jongkok dalam Lari Jangka Pendek

Bagian sebelumnya telah dijelaskan bahwa ada setidaknya tiga macam jenis start jongkok yang dapat anda gunakan untuk memulai lari jarak pendek. Ketiga macam jenis start jongkok tersebut antara lain adalah start pendek, start menengah, dan start panjang. Berikut ini anda dapat mengetahui bagaimana jenis start tersebut dapat anda gunakan dengan teknik yang benar.

Macam Start Pendek – Short Start / Bunch Start Type

Perhatikan bagaimana anda melakukan sikap permulaan pada jenis start yang pertama ini. Artinya, ada sikap awal saat memulai start pendek. Sikap awal tersebut antara lain dimulai dengan berdiri tegak dan kedua kaki anda masih dalam posisi rapat. Setelah itu, posisikan tangan anda di kedua sisi badan anda dengan tatapan fokus ke depan. Saat akan memasuki fase posisi start, mulailah dengan melangkahkan salah satu kaki anda ke depan. Umumnya kaki yang dijadikan tumpuan depan adalah kaki kiri. Posisi jari kaki adalah lurus ke depan. Kemudian posisikan juga posisi kaki belakang menjadi sejajar dengan tumit kaki yang ada di posisi di depan. Perlu anda perhatikan bahwa ada jarak ideal antara kaki yang di depan dan yang diposisi belakang, yaitu sekitar satu kepal tangan jauhnya. Terakhir, posisikanlah kedua lengan anda sebagai tumpuan dengan posisi yang lurus dengan bahu. Jari jari anda tetap dalam posisi rapat dan ibu jari anda terbuka persis sebelum melewati garis start.

Macam Start Menengah – Medium Start Type

Sama seperti teknik start sebelumnya, macam macam start dalam lari jarak pendek berikutnya perlu menyiapkan sikap permulaan yang tepat. Kali ini jenis start menengah yang akan dilakukan. Posisi siap di awal pertandingan adalah sama yaitu dengan berdiri tegak menggunakan kedua kaki yang dalam posisi persis rapat. Fokuskan pandangan anda ke depan dan kedua tangan lurus di sisi badan anda. Tepat setelah pertandingan akan dimulai, anda dapat memosisikan tubuh pada posisi start melayang. Jangan lupa untuk mengangkat sedikit posisi bagian bawah tubuh anda sebagai start melayang tanda dilakukan.

Macam Start Panjang – Long Start Type

Start panjang adalah satu lagi dari macam macam start dalam perlombaan lari yang juga dilakukan dengan diawali dengan posisi sikap awal. Tidak ada bedanya, hanya saja posisi kaki belakang dan kaki depan akan memiliki sedikit perbedaan dengan start pendek ataupun start menengah. Jadi pastikanlah untuk mengetahui dahulu bagaimana anda bisa melakukan macam macam start jongkok ini sebelum anda memutuskan akan menggunakannya atau tidak. Setelah mengetahui macam start yang anda perlu lakukan, maka pelajarilah langkah selanjutnya mengenai teknik saat mendengar aba-aba start.

Teknik Fokus Mendengar Aba-aba Start Jongkok

Setelah melakukan start jongkok apapun macamnya, maka anda perlu menyiapkan diri dalam mendengar aba—aba start. Aba-aba adalah istilah yang digunakan untuk menjelaskan tanda dimulainya pertandingan. Berikut ini teknik yang perlu diketahui dalam beberapa langkah saat mendengar aba-aba. Aba aba yang pertama adalah “bersedia,” jenis aba-aba ini adalah aba aba yang pertama yang menandakan pemain harus siap sebelum berlari. Pelari melakukan posisi bersedia dengan menggunakan satu kaki untuk melangkah ke depan, namun masih di belakang garis start. Saat melangkah, pemain pun dapat memosisikan tubuh sedikit berjongkok sesuai dengan ketentuan start yang digunakan apakah start pendek, start menengah, ataukah start panjang. Jika sudah, maka posisikan kedua tangan anda hingga ke jari-jarinya ke bagian tengah. Aba-aba setelah “bersedia” adalah aba-aba “siap”, pada aba-aba ini ada perubahan posisi yang perlu dilakukan oleh pemain yaitu dengan melakukan pengangkatan bokong ke posisi yang menjadi lebih tinggi dari panggul dan bahu pemain. Selanjutnya, posisikan juga kepala pemain menjadi menunduk dan tidak lagi tegang. Aba-aba terakhir biasanya adalah suara pistol yang ditembakkan. Jika tidak ada pistol, biasanya akan diganti dengan aba-aba “ya.” Saat aba-aba “ya,” pemain dapat segera melakukan dorongan kaki belakang untuk lari melangkah ke depan dan memulai pertandingan. Demikianlah start yang baik dapat dilakukan.
