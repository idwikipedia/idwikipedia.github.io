URL (Uniform Resource Locator) adalah rangkaian karakter menurut suatu format standar tertentu, yang digunakan untuk menunjukkan alamat suatu sumber seperti dokumen dan gambar di Internet. URL pertama kali diciptakan oleh Tim Berners-Lee pada tahun 1991 agar penulis-penulis dokumen dokumen dapat mereferensikan pranala ke World Wide Web. Sejak 1994, konsep URL telah dikembangkan menjadi istilah Uniform Resource Identifier (URI) yang lebih umum sifatnya.

DOMAIN menurut Wikipedia Nama Domain (domain name) adalah nama unik yang diberikan untuk mengidentifikasi nama server komputer seperti web server atau email server di jaringan komputer ataupun internet. Nama domain berfungsi untuk mempermudah pengguna di internet pada saat melakukan akses ke server, selain juga dipakai untuk mengingat nama server yang dikunjungi tanpa harus mengenal deretan angka yang rumit yang dikenal sebagai IP address. 34. tuliskan 3 bagian dari url​


Jawaban:

Terdapat 3 bagian yang umumnya kita jumpai pada sebuah URL, tiga bagian ini adalah Protokol, Hostname, dan Path.

Penjelasan:

semofa membantu

pengertian:

URL (Uniform Resource Locator) adalah alamat unik yang digunakan untuk mengakses sumber daya (resources) tertentu di internet. Adapun sumber daya yang dimaksud disini adalah semua sumber daya yang tersedia di internet, hal ini bisa berupa halaman web (document HTML), Gambar, Video dan berbagai sumber daya lainnya.

Terdapat 3 bagian yang umumnya kita jumpai pada sebuah URL, tiga bagian ini adalah Protokol, Hostname, dan Path. 

Protokol

Protokol adalah bagian URL yang mendefinisikan "bagaimana proses komunikasi akan dilakukan". Sederhananya, protokol adalah seperangkat aturan yang menjelaskan bagaimana proses langkah demi langkah komunikasi antar dua atau lebih komputer dilakukan. Jaringan akan mengikuti aturan ini untuk dapat mentransmisikan data dengan benar. Adapun protokol standar yang sering kita jumpai adalah HTTP (Hypertext Transfer Protocol) dan HTTPS (HTTP Secure). Selain 2 protokol ini, terdapat beberapa protokol lainnya yang digunakan pada URL, sebut saja misalnya mailto: yang mengintruksikan browser untuk membuka program email default pada perangkat client. 

Untuk kegiatan pada browser, Prokokol merupakan bagian opsional. Dengan kata lain, kita bisa mengetikkan URL dengan atau tanpa protokol pada address bar browser. Dalam hal ini, jika protokol tidak didefinisikan, browser akan menggunakan protokol default (http).

Hostname

Hostname (bisa juga disebut sebagaidomain name) adalah bagian yang menunjukkan alamat utama website yang menerbitkan sumber daya bersangkutan.

Path

Menunjukkan lokasi file yang diakses.
