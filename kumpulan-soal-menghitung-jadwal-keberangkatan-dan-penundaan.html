25. Jadwal keberangkatan bus rajawali tanjung balai medan


Untuk jurusan Medan -Tanjung Balai saat ini terdapat beberapa moda tranportasi, antara lain: bus dan juga kereta api, saat ini kereta api sudah memenuhi ke-lima indikator kenyamanan dan fasilitas tersebut.

Jadwal keberangkatan bus Rajawali pada pukul 2 siang berangkat dari Binjai menuju Tanjung Balai.

Pembahasan:

Untuk jurusan Medan -Tanjung Balai terdapat beberapa moda tranportasi, antara lain adalah: bus dan juga kereta api, saat ini kereta api sudah memenuhi ke-lima indikator kenyamanan dan fasilitas tersebut. Seperti fasilitas yang sangat lengkap dimulai dari stasiun yang bersih serta keadaan didalam gerbong yang bersih dan nyaman, harga tiketnya yang dapat dikatakan murah, dan sesuai jadwal. Pihak PT KAI juga bekerjasama dengan asuransi jasa raharja, serta kemampuan masinis dalam mengendarai kereta api pun sudah tak perlu diragukan lagi.

 Ada beberapa bus yang digunakan dalam perjalanan Medan-Tanjungbalai yaitu, Almasar, Rajawali dan bus KUPJ. Bus Rajawali dan KUPJ hingga saat ini belum memenuhi ke-lima indikator tersebut, supir dalam mengendarai bus-bus ini sering dijumpai ugal-ugalan dalam mengemudi, seperti memotong jalan tanpa memikirkan konsekuensi yang akan diterima nantinya.

38. Tulislah 5 kalimat berdasarkan jadwal keberangkatan kereta api di atas! Tolong secepatnya yah... soalnya tugas adikku ini akan dikumpulkan hari ini...​


Berdasarkan pertanyaan di atas maka kalimat mengenai jadwal keberangkatan kereta adalah sebagai berikut:

Jadwal keberangkatan kereta api pada hari senin adalah pukul 06.00.Jadwal keberangkatan kereta api pada hari selasa adalah pukul 06.00.Jadwal keberangkatan kereta api pada hari rabu adalah pukul 06.00.Jadwal keberangkatan kereta api pada hari rabu adalah pukul 06.00.Jadwal keberangkatan kereta api pada hari senin sampai hari minggu adalah sama, yaitu pukul 06.00.

Pembahasan:

Pada soal diatas termasuk kedalam menyusun atau membuat kalimat dengan menggunakan kata-kata yang telah disediakan. Kalimat merupakan gabungan dari beberapa kata. Setiap kalimat yang satu dengan kalimat yang lainnya dihubungkan dengan sebuah konjungsi. Adapun kumpulan dari beberapa kalimat disebut dengan paragraf. Latihan membuat kalimat dalam kelas rendah dapat melatih keterampilan anak untuk berpikir sejak dini. Anak akan belajar bagaimana cara untuk berpikir, menganalisis dan menyusun. Kegiatan menulis dan menyusun kata dapat melatih kemampuan literasi anak. Pada kelas rendah, guru dapat menggunakan benda-benda konkret sebagai media pembelajaran untuk membantu siswa atau sebagai stimulus agar siswa dapat menemukan kata-kata dengan cara membayangkan benda yang ada di depannya.
