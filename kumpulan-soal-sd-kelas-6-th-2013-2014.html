12. urusan pemerintah pusat dan daerah dalam uu no.23 th.2014 adalah(?).Tolong jawab segera,soalnya dikumpulkan besuk!"makasih"


Berdasarkan UU Nomor 23 tahun 2014 klasifikasi urusan pemerintahan terdiri dari 3 urusan yakni urusan pemerintahan absolut, urusan pemerintahan konkuren, dan urusan pemerintahan umum. Urusan pemerintahan absolut adalah Urusan Pemerintahan yang sepenuhnya menjadi kewenangan Pemerintah Pusat. Urusan pemerintahan konkuren adalah Urusan Pemerintahan yang dibagi antara Pemerintah Pusat dan Daerah provinsi dan Daerah kabupaten/kota. Urusan pemerintahan umum adalah Urusan Pemerintahan yang menjadi kewenangan Presiden sebagai kepala pemerintahan.

18. soal UN 2013/2014 no.13


maaf harus di poto :)
clik gambar biar liat penuh :)Diket dan dit:
*n NaOH = 100 mL x 0,008 M = 0.8 mmol
*n CH3COOH = 100 mL x 0,008 M = 0,8 mmol
*Ka = 10^-5
*Ksp Fe(OH)2 = 1,6 x 10^-15
*[Fe2+] = ? M

A. Mencari nilai [OH-]
--> Soal UN ini mengaitkan 2 bab (kyk SBMPTN, ya), yaitu hidrolisis garam dan hasil kali kelarutan (Ksp), sehingga:
     CH3COOH + NaOH --> CH3COONa + H2O
m        0.8           0,8
b        -0.8          -0.8              +0.8          +0.8
s           -              -                 0.8

[G] = 0.8 mmol / 200 mL = 4 x 10^-3

Bersisa garamnya sebesar 0,8 mmol, masukkan ke rumus hidrolisis:
[OH-] = √Kw x [G]/Ka
         = √10^-14 x 4 x 10^-3 / 10^-5
         = 2 x 10^-6

B. Mencari [Fe2+] dari Fe(OH)2
--> Untuk mencarinya, buat reaksi ionisasi Besi (II) hidroksida tsb:
Fe(OH)2 ⇄ Fe2+ + 2OH-

Ksp = [Fe2+] [OH-]^2
1,6 x 10^-15 = [Fe2+] [2 x 10^-6]^2
[Fe2+] = 1,6 x 10^-15 / 4 x 10^-12
           = 0.4 x 10^-3
           = 4 x 10^-4 Molar (D)

Diketahui = Hasil Panen Kedelai

Tahun 2013 = 100 kgTahun 2014 = 75 kg Tahun 2015 = 100 kg Tahun 2016 = 150 kg Tahun 2017 = 175 kg

Ditanya = Rata-rata tahun 2013-2017 = … ?

Jawab =

[tex]rata - rata = \frac{seluruh \: hasil \: panen}{jumlah \: tahun} [/tex]

[tex] = \frac{100 + 75 + 100 + 150 + 175}{5} [/tex]

Seluruh hasil panen = 100+75+100+150+175 = 175+100+150+175 = = 275+150+175 = 425+175 = 600

[tex] = \frac{600}{5} = 120[/tex]

Jadi, rata-rata panen kedelai tahun 2013-2017 adalah 120 kedelai
