20. Apa syarat agar soal limit boleh dikerjakan dg turunan?


Jawab:

Limit trigonometri adalah nilai terdekat suatu sudut pada fungsi trigonometri. Perhitungan limit fungsi trigonometri bisa langsung disubtitusikan seperti limit fungsi aljabar tetapi ada fungsi trigonometri yang harus diubah terlebih dahulu ke identitas trigonometri untuk limit tak tentu yaitu limit yang apabila kita langsung subtitusikan nilai nya bernilai 0, atau bisa juga untuk limit tak tentu tidak harus memakai identitas tetapi memakai teorema limit trigonometri dan ada juga yang memakai identitas dan teorema. Jadi, apabila suatu fungsi limit trigonometri di subtitusikan nilai yang paling mendekati nya menghasilkan dan maka kita harus menyelesaikan dengan cara lain.

JAWABAN

⇒ [tex]\frac{1}{2}[/tex]

PENYELESAIAN

⇒ [tex]\lim_{x \to \infty}\frac{(x^5+6x-10x^3)(x+1)}{(x^4+5x^2)(2x^2-3)}[/tex]

⇒ [tex]\lim_{x \to \infty}\frac{(x^5(1+\frac{6}{x^4}-\frac{10}{x^2})x(1+\frac{1}{x}) }{x^4(1+\frac{5}{x^2})x^2(2-\frac{3}{x^2}) }[/tex]

⇒ [tex]\lim_{n \to \infty}\frac{(\frac{6}{x^4}-\frac{10}{x^2}+1)(\frac{1}{x}+1) }{(\frac{5}{x^2}+1)(-\frac{3}{x^2}+2) }[/tex]

Sudah jelas bahwa antara limit dengan turunan saling berkaitan.

Sementara hubungan limit dengan integral, y kalau dalam integral limit itu dipakai untuk menentukan rumus luas daerah dibawah kurva yang kita biasa pakai sehari - hari, ini disebut juga metode integral Riemann. 33. materi : turunan fungsi limit trigonometri​


Nilai dari limₓ→₀(1-cos3x)/(1-cosx) =

9

opsi D

Pembahasan:

Aturan L'Hospital atau Turunan:

limₓ→ₐ f(x) / g(x) =

limₓ→ₐ f'(x) / g'(x)

syarat: limₓ→ₐ f'(x) / g'(x) ada.

Limit untuk x = 0 berlaku (sin(px)) / x ) = p

Turunan:

f(x) = cos(ax)

f'(x) = -sin(ax) × a

Diketahui:

Ditanya:

Nilai dari limₓ→₀(1-cos3x)/(1-cosx) =?

Dijawab:

Untuk mencari nilai limit tersebut pertama karena tidak bisa dijabarkan penyebutnya maka menggunakan turunan di pembilangnya dan turunan di penyebutnya.

lalu Gunakan aturan:

Limit untuk x = 0 berlaku (sin(px)) / x ) = p

f(x) = 1 - cos3x

f'(x) = -(-sin3x)3

= 3sin3x

g(x) = 1 - cosx

g'(x) = - (-sinx)

= sinx

f'(x) / g'(x) =

3sin3x / sinx

limit untuk x = 0 berlaku (sin3x / x ) = 3

limₓ→₀ 3sin3x / sinx =

3 × 3 =

9

Jadi Nilai dari limₓ→₀(1-cos3x)/(1-cosx) =

9

40. tentukan turunan fungsi 4x³dengan menggunakan limit fungsi​


Materi : Turunan, dan Limit Fungsi

Lim h => 0 ( f(x+h)³ - f(x) )/h

____________________/

= ( 4( x + h )³ - 4x³ )/h

= ( 4( x³ + 3x²h + 3xh² + h³ ) - 4x³ )/h

= ( 4x³ + 12x²h + 12xh² + 4h³ - 4x³ )/h

= ( 12x²h + 12xh² + 4h³ )/h

= 12x² + 12xh + 4h²

Lim h => 0

= 12x²
