4. Jelaskan definisi tentang teorema faktor


teorema faktor : sebuah pernyataan biimplikasi/implikasi dua arah
( x - k ) adalah faktor dari f(x)  jika dan hanya jika f(k) = 0
teorema faktor adalah sebuah pernyataan biimplikasi atau implikasi dua arah. 
contoh : (x-b) adalah faktor dr f(x) jika dan hanya jika f(b)= 0
bukti:
misalnya (x-b) merupakan faktor dr f(x), maka
f(x)= (x-b).H(x) .... (1)
H(x) itu adalah hasil bagi , subsitusikan nilai x=b kepersamaan 1
f(b)=(b-b).H(b)... (1)
f(b)= 0
jadi kebukti (x-b) adalah faktor dr f(x) maka f(b)=0

27. [25 Poin] Materi Teorema Polinomial ​


Polinomial

sisa = f(x) = ax + b

dibagi (3x + 2) → sisa 1

f(-2/3) = 1

-2/3 a + b = 1 ... (1)

dibagi (2x - 5) → sisa 20

f(5/2) = 20

5/2 a + b = 20 ... (2)

dibagi 6x² - 11x - 10

6x² - 11x - 10 = (2x - 5)(3x + 2)

(1) - (2)

-2/3 a - 5/2 a = 1 - 20

-19/6 a = -19

a = 6

5/2 a + b = 20

5/2 × 6 + b = 20

15 + b = 20

b = 5

Sisa bagi = ax + b = 6x + 5

Diketahui

p(x) = 6x² - 11x -10 = (3x + 2)(2x - 5)

p(-2/3) = 1

p(5/2) = 20

Misalkan

h(x) = hasil

Persamaan 1

p(x) = h(x) × (6x² - 11x - 10) + (ax + b)

p(-2/3) = h(-2/3) × (6(-2/3)² - 11(-2/3) + (a(-2/3) + b)

1 = h(-2/3) × 0 + (-2/3)a + b

1 = (-2/3)a + b

b = 1 + (2/3)a (i)

Persamaan 2

p(x) = h(x) × (6x² - 11x - 10) + (ax + b)

p(5/2) = h(5/2) × (6(5/2)² - 11(5/2) - 10) + (a(5/2) + b)

20 = h(5/2) × 0 + (5/2)a + b

20 = (5/2)a + b (ii)

Substitusi (i) ke (ii)

20 = (5/2)a + 1 + (2/3)a

20 - 1 = (15/6 + 4/6)a

19 = (19/6)a

a = 19 × 6/19

a = 6

Substitusi a ke (i)

b = 1 + (2/3)(6)

b = 1 + 4

b = 5

Sisa pembagian p(x)

ax + b

= 6x + 5
