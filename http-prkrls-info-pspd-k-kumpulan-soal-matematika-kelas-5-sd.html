7. tolooonggg info cara cari K


Jawab:

nilai K = 1

Penjelasan dengan langkah-langkah:

diketahui :
[tex]\frac{2}{x } + \frac{1}{y} = -1\\\frac{1}{x } + \frac{3}{y} = 7\\\\2x + 6y = K[/tex]

misalkan :
[tex]\frac{1}{x} = a\\[/tex]
[tex]\frac{1}{y} = b[/tex]
2a + b = -1...Pers 1
a + 3b = 7...Pers 2

ditanya : K = ?

gunakan metode substitusi dan eliminasi
2a + b = -1 ║×1║
a + 3b = 7 ║×2║

2a + b = -1
2a + 6b = 14
------------------ ( - )
-5b = -15
b = [tex]\frac{-15}{-5}[/tex]
b = 3

substitusi b ke persamaan 2
a + 3b = 7
a + 3 (3) = 7
a + 9 = 7
a = 7 - 9
a = -2

maka,
[tex]\frac{1}{x} = a[/tex]
[tex]\frac{1}{x} = -2\\\\\frac{1}{-2} = x\\\\x = \frac{-1}{2}[/tex]

[tex]\frac{1}{y} = b[/tex]
[tex]\frac{1}{y} = 3\\\\\frac{1}{3} = y[/tex]

maka,
[tex]2x + 6y = K\\2(\frac{-1}{2} ) + 6 (\frac{1}{3} ) = K\\-1 + 2 = K\\K = 1[/tex]

40. Bagaimana ya kakak2, abang2, menjelaskan pelajaran mengenai sudut dengan gampang utk pelajaran kelas 5 beserta contoh soalnya juga. Beri info donk kakak2 & abang² sekalian. Trimakasih..


Contoh Soal :

1. Sudut apakah yang terdapat pada bangun persegi dan persegi panjang?

Jawab :

Sudut yang terdapat pada persegi dan persegi panjang adalah sudut siku-siku karena besar sudutnya 90 derajat

2. Berapa besar sudut uang dibentuk jarum jam saat menunjukkan pukul 03.00?

Jawab :

Besar sudut tiap jam =

360 : 12

= 30 derajat

Besar sudut jarum jam

= 30 derajat x 3

= 90 derajat

3. Sudut 150 derajat termasuk sudut....

Jawab :

Sudut 150 derajat termasuk sudut tumpul karena

[tex] 150 > 90[/tex]

4. Alat untuk mengukur besar sudut adalah....

Jawab :

Alat untuk mengukur besar sudut adalah busur derajat

Rangkuman Mini :

a. Besar seluruh sudut pada segitiga adalah 180 derajat

b. Sudut bangun persegi dan persegi panjang adalah 90 derajat

c. Sudut lurus = 180 derajat

d. Sudut siku siku = 90 derajat

e. Sudut lancip = lebih kecil dari 90 derajat

f. Sudut tumpul = lebih besar dari 90 derajat

g. Titik sudut adalah ujung pertemuan dari kedua garis pembentuk sudut

Catatan :

Mengukur sudut harus menggunakan busur derajat.

Tips :

1. Pilihlah busur derajat yang bagian tengahnya tidak berlubang supaya mudah untuk mengukurnya

2. Buatlah rangkuman/carilah cara mudah untuk menghafal pelajaran tentang sudut, bosa dengan mengubah lirik lagu menjadi rangkuman pelajaran supaya lebih mudah dan lebih asyik untuk mempelajarinya

3. Jika tidak ada busur derajat, kamu dapat menentukan jenis sudut untuk mengamatinya
