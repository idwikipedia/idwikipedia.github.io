3. perbedaan format exfat dengan fat32??... mohon penjelasannya


Format exfat adalah upgrade dari format fat32. Format exfat mendukung semua versi windows dan versi mac os (yang modern) serta linux (dengan membutuhkan software tambahan).

Format fat32 adalah upgrade dari format fat16. Format fat32 saat ini masih dipakai untuk flash drive atau media penyimpanan lain. Namun jarang dipakai lagi untuk komputer yang modern, sebab kini format yang sering dipakai ialah ntfs.

13. PETUNJUK UMUM 1. Gambarkan dan jelaskan hubungan antara Threat, Risk dan Countermeasure 2. Berbicara tentang administrator ada 2 jenis type administrator, yaitu admin local dan admin jaringan. Tolong jelaskan perbedaan masing-masing dari admin tersebut? dan mana yang lebih tinggi mempunyai hak akses 3. Di dalam system windows ada satu konsep untuk memanage security secara tercental. Sebutkan sistem apa dan bagaimana konsep nya 4. Sebutkan perbedaan antara ntfs, fat32, fat, swap


Penjelasan:

Tentu, berikut adalah jawaban untuk setiap pertanyaan Anda:

1. **Hubungan antara Threat, Risk, dan Countermeasure:**

  - **Threat (Ancaman)** adalah potensi atau kemungkinan terjadinya peristiwa yang dapat merusak atau membahayakan sistem atau informasi. Ancaman dapat datang dari berbagai sumber, seperti virus komputer, serangan siber, atau bencana alam.

  - **Risk (Risiko)** adalah tingkat kerentanan sistem atau informasi terhadap ancaman tertentu. Ini mencakup seberapa besar dampak yang mungkin terjadi jika ancaman tersebut terjadi dan seberapa besar kemungkinan ancaman tersebut terjadi.

  - **Countermeasure (Tindakan Pengamanan)** adalah langkah-langkah atau strategi yang diambil untuk mengurangi risiko yang ditimbulkan oleh ancaman. Ini termasuk implementasi kebijakan keamanan, perangkat lunak keamanan, pemantauan, dan tindakan lainnya untuk melindungi sistem atau informasi.

2. **Perbedaan antara Admin Local dan Admin Jaringan:**

  - **Admin Local (Administrator Lokal)** adalah seseorang yang memiliki hak akses dan kendali atas komputer atau server tertentu. Mereka memiliki kekuasaan untuk mengelola pengguna, perangkat keras, dan perangkat lunak di komputer tersebut.

  - **Admin Jaringan (Administrator Jaringan)** adalah seseorang yang bertanggung jawab atas manajemen dan pengelolaan jaringan komputer yang lebih besar. Mereka memiliki hak akses untuk mengelola perangkat jaringan, router, server, dan kebijakan keamanan jaringan. Admin jaringan umumnya memiliki hak akses yang lebih tinggi dibandingkan admin lokal karena mereka mengelola seluruh jaringan.

3. **Manajemen Keamanan Sentral di Windows:**

  - Konsep manajemen keamanan sentral yang umum digunakan di Windows disebut **Active Directory**. Active Directory adalah layanan direktori yang memungkinkan organisasi untuk mengelola dan mengotentikasi pengguna, komputer, dan sumber daya jaringan dalam satu tempat terpusat. Dengan Active Directory, administrator dapat mengatur kebijakan keamanan, hak akses, dan manajemen pengguna secara sentral di seluruh jaringan.

4. **Perbedaan antara NTFS, FAT32, FAT, dan Swap:**

  - **NTFS (New Technology File System)** adalah sistem file yang lebih canggih dan kuat digunakan dalam sistem operasi Windows. Ini mendukung fitur keamanan yang lebih baik, pengelolaan disk yang lebih baik, dan batasan ukuran file yang lebih besar dibandingkan dengan sistem file FAT.

  - **FAT32 (File Allocation Table 32-bit)** adalah sistem file yang lebih tua yang digunakan dalam sistem operasi Windows. Ini memiliki batasan ukuran file yang lebih kecil dibandingkan NTFS, tetapi lebih kompatibel dengan berbagai sistem operasi.

  - **FAT (File Allocation Table)** adalah sistem file yang lebih tua dari FAT32. Ini memiliki batasan ukuran file dan partisi yang lebih rendah dan biasanya digunakan dalam sistem operasi DOS dan Windows awal.

  - **Swap** biasanya merujuk pada ruang swap dalam sistem operasi, terutama dalam sistem berbasis Unix/Linux. Ini adalah ruang yang digunakan untuk menyimpan data yang dimuat atau dipindahkan dari RAM saat RAM telah mencapai kapasitas maksimum. Ini membantu memastikan kinerja sistem yang stabil.
