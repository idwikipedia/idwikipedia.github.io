cara mengatur penyimpanan aplikasi android ke memori eksternal SD Card secara default :
√ Buka menu Settings atau Pengaturan, lalu pilih Penyimpanan.
√ Setelah masuk ke pilihan Penyimpanan, maka scrolling ke bagian bawah Penyimpanan, pilih Write Disk Default.
√ Pada Write Disk Default, atur pilihan penyimpanan perangkat ke memori eksternal atau SD Card. Selesai.
